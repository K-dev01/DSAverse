# DSA Verse

A premium DSA revision website built with React, Vite, Tailwind CSS, React Router, Framer Motion, and Lucide React.

## Project Structure

- `src/components/` — reusable UI primitives such as buttons, cards, navbar, footer, and scroll helpers
- `src/pages/` — placeholder pages for Home, STL Library, Patterns, Templates, Cheat Sheets, Interview Tricks, Downloads, and About
- `src/layouts/` — shared page layout structure
- `src/hooks/` — theme context and dark/light toggle management
- `src/data/` — navigation link definitions and shared content data
- `src/styles/` — global Tailwind CSS styles
- `src/utils/` — utility helpers

## Available Scripts

- `npm install` — install dependencies
- `npm run dev` — start the development server
- `npm run build` — build the production assets

## Notes

- Dark/light theme toggle is supported through a theme provider
- Routing is powered by `react-router-dom`
- The UI is designed for an elegant revision experience, not a coding judge
